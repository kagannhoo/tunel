## OSINT Tools

1. **Sherlock Project** — Username search across social networks.  
   https://github.com/sherlock-project/sherlock

2. **WhatsMyName** — Micah Hoffman (WebBreacher). Centralized JSON database for username enumeration.  
   https://github.com/WebBreacher/WhatsMyName
